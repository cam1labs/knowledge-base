# Knowledge Base

Esta base de conhecimento reúne o contexto profissional, processos, projetos, padrões de documentação e regras de negócio utilizados no meu dia a dia.

Objetivos:

- Evitar repetição de contexto entre conversas.
- Garantir consistência nas respostas das IAs.
- Centralizar decisões importantes.
- Facilitar onboarding de novos projetos.

## Estrutura

00-profile → informações sobre mim e minha forma de trabalhar

01-company → empresa, produtos e contexto

02-product → processos, templates e convenções

03-projects → contexto de cada projeto

04-business-rules → regras compartilhadas

05-decisions → decisões importantes (ADRs)

templates → modelos reutilizáveis
